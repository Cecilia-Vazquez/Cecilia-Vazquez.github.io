Tecnologías Web

HTML:

(Hypertetx Markup Lenguaje) Lenguaje de Marcado para Texto.Hace referencia al lenguaje de marcado para la elaboración de páginas web. Es un estándar que sirve de referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros.

  • Las paginas de HTML son docuementos de texto
  • HTML usa elemntos para definir la estructura de la pagina
  • Los elemntos se definen por etiquetas encerradas en parentesis angulares < y >
  • Las etiquetas definen el contenido de un elemento
  • Exiten etiquetas de apertura <entiqueta> y de cierre </etiqueta>.
  • El nombre de la etiqueta debe ser el mismo y se distinge a la etiqueta de cierre con una diagonal.   <etiqueta>contenido </etiqueta>.
  • Las etiquetas pueden tener atributos los cuales se especifican ene la etiqueta de inicio.
  • Los atributos contienen informacion adiocional acerca del contenido.   <etiqueta atributo="valor"> contenido </etiqueta>
